Features

Leaderboards

We provide leaderboards for a vast set of tasks ranging from text classification to natural language Understanding with the aim of standardizing the evaluation of the performance of Turkish Natural Language Processing models. Through this platform, researchers are given the opportunity to access the performance of models and their respective open source implementations on a single platform.

Fair On-cloud Evaluation

All the models are evaluated in a fair environment by an extensively tested and open source benchmarking tools. All of the evaluations are made on unannotated datasets with an automated cloud-based environment. A great range of datasets are provided on Turkish Data Depository. Researchers can submit datasets they developed and organise challenges to push the frontiers of the science.

Open Source Implementations of the State-of-the-art

To ensure that the submitted model outputs are valid, researchers are required to share an open source implementation of their models that will be publicly visible to the researchers worldwide. With the published open source implementations, the inspiration and informativeness of the previous works are maximized.

Baselines

Currently, baseline models for spell-checking, text classification, language modeling, named-entity recognition, machine translation, natural language inference, tokenization and part-of-speech tagging are provided.

Structured Set of Datasets

Hand by hand with Turkish Data Depository project, an extensive set of datasets are provided to the researchers with detailed documentations and in a easy-to-use manner. Under this project; trnews-64, a character language modeling dataset that contain 64 million words of news articles and columns, and trwiki-67,a language modeling dataset that contain 67 million words of raw wikipedia articles, datasets are constructed and there is ongoing effort on many more.

Contests

Mukayese provides the standardized platform reserachers are in need of where they can publish the datasets they created and organise contests thereon.
